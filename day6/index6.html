    Cú pháp 1:
        INSERT INTO table_name(colum1, column2, column3, ..)
        VALUES (values1, values2, values3,...);
    Cú pháp 2:
        INSERT INTO table_name
        VALUES (values1, values2, values3, ...);
    Ví dụ:
    Thêm 1 bản ghi mới vào bảng customers ( cột id thì bỏ qua vì id sẽ tự sinh)
